# Курсова робота: «Кістки»
C++ / SFML Application
Опис проєкту

«Кістки» — це графічний навчальний застосунок, розроблений мовою C++ з використанням бібліотеки SFML.
Програма призначена для вивчення кісткової системи людини в інтерактивному форматі.

Застосунок дозволяє переглядати типи кісток, їхню структуру, функції та додаткову інформацію через зручний графічний інтерфейс.

Проєкт виконаний у межах курсової роботи з дисципліни програмування.

Функціональні можливості

Графічний інтерфейс на основі SFML

 Інформаційні вікна з описом

 Відображення зображень кісток

 Навігація між розділами

 Кнопка виходу з програми

 Використані технології
